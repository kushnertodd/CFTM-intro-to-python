This is a python program to find all possible Wordle words given the available letters and letters that have fixed positions

Install:

Python 3.10 from Microsoft Store

To run:

python wordle-dict letters fixed

where:

letters = the letters available to use in a word
fixed = letters in fixed positions
        a 5-letter string where each character is a letter in a fixed position a '.' if not fixed
  for example, if 't' must be in position 2 and 'o' must be in position 4, fixed = '.t.o.'
